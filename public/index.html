M365 / Outlook Setup — 5 Steps

REQUIRED

SAM's M365 integration code is deployed and ready. You just need to register an Azure app and drop in three values.

1. Go to Azure Portal → Entra ID → App Registrations → New Registration
Open Azure Portal →
2. Name it "SAM-ChiefOfStaff" — Single tenant — Register
Copy the Application (Client) ID and Directory (Tenant) ID from Overview page
3. API Permissions → Add: Mail.Read, Mail.Send, Mail.ReadWrite, Calendars.ReadWrite
Click "Grant admin consent" after adding permissions
4. Certificates & Secrets → New Client Secret → Copy the Value
Save this immediately — it only shows once
5. Give SAM the three values (Client ID, Tenant ID, Client Secret)
SAM stores them in Netlify environment variables and the Anthropic vault
